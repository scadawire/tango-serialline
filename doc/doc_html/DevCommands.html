TANGO Device Server
Device Commands Description


Revision: 1.3 - Author: perez



1 - State

  • Description: Command to read the device state. The method dev_state() override a DeviceImpl class virtual method.
     
  • Argin:
    Tango::DEV_VOID     :
     
  • Argout:
    Tango::DEV_STATE     : State Code
     
  • Command allowed for:

     


2 - Status

  • Description: Command to read the device status. The method dev_status() override a DeviceImpl class virtual method.
     
  • Argin:
    Tango::DEV_VOID     :
     
  • Argout:
    Tango::CONST_DEV_STRING     : Status descrition
     
  • Command allowed for:

     


3 - DevSerWriteString

  • Description: Write a string of characters to a serial line and return the number of characters written.
     
  • Argin:
    Tango::DEV_STRING     : String to write
     
  • Argout:
    Tango::DEV_LONG     : Number of characters written
     
  • Command allowed for:

     


4 - DevSerWriteChar

  • Description: Write N characters to a serial line and return the number of characters written.
     
  • Argin:
    Tango::DEVVAR_CHARARRAY     : Characters to write
     
  • Argout:
    Tango::DEV_LONG     : Number of characters written
     
  • Command allowed for:

     


5 - DevSerWriteRead

  • Description: This method permit to send a request to a device throw the serial line and returns the response of the device. The commands write and read don't return until they have not finished.
     
  • Argin:
    Tango::DEV_STRING     : command to write on the port com
     
  • Argout:
    Tango::DEV_STRING     : response of the device behind the serial line
     
  • Command allowed for:

     


6 - DevSerReadNChar

  • Description: Read a string of N characters from the serial line If there are no characters to be read returns an empty string.
     
  • Argin:
    Tango::DEV_LONG     : number of characters to read on the serial line
     
  • Argout:
    Tango::DEV_STRING     : pointer to the string read updated
     
  • Command allowed for:

     


7 - DevSerReadRaw

  • Description: read a string from the serialline device in mode raw (no end of string expected, just empty the entire serialline receiving buffer).
     
  • Argin:
    Tango::DEV_VOID     : no argin
     
  • Argout:
    Tango::DEV_STRING     : pointer to the string read updated
     
  • Command allowed for:

     


8 - DevSerReadLine

  • Description: read a string from the serialline device in mode line (up to and including the character specified by the NewLine property
     
  • Argin:
    Tango::DEV_VOID     : no argin
     
  • Argout:
    Tango::DEV_STRING     : pointer to the string read updated
     
  • Command allowed for:

     


9 - DevSerSetParameter

  • Description:
     
  • Argin:
    Tango::DEVVAR_LONGARRAY     : device parameters in pair
     
  • Argout:
    Tango::DEV_VOID     : no argout
     
  • Command allowed for:

     


10 - DevSerSetTimeout

  • Description: This command sets the new timeout.
     
  • Argin:
    Tango::DEV_SHORT     : Value of the timeout to set
     
  • Argout:
    Tango::DEV_VOID     : no argout
     
  • Command allowed for:

     


11 - DevSerSetParity

  • Description: Sets the new parity of the serial line.
     
  • Argin:
    Tango::DEV_SHORT     : The new parity to set
     
  • Argout:
    Tango::DEV_VOID     : no argout
     
  • Command allowed for:

     


12 - DevSerSetCharLength

  • Description: Sets the new charlength. It can be 8, 7, 6, 5.
     
  • Argin:
    Tango::DEV_SHORT     : The new charlength to set
     
  • Argout:
    Tango::DEV_VOID     : no argout
     
  • Command allowed for:

     


13 - DevSerSetStopbit

  • Description: Sets the new stop bit (0=none, 1=one stop, 2=1.5stop bit).
     
  • Argin:
    Tango::DEV_SHORT     : The new stopbit to set
     
  • Argout:
    Tango::DEV_VOID     : no argout
     
  • Command allowed for:

     


14 - DevSerSetBaudrate

  • Description: Sets the new baudrateof the serial line up to 9600 max.
     
  • Argin:
    Tango::DEV_SHORT     : The new baudrate to set
     
  • Argout:
    Tango::DEV_VOID     : no argout
     
  • Command allowed for:

     


15 - DevSerSetNewline

  • Description: The new ending character in hexa.
     
  • Argin:
    Tango::DEV_SHORT     : The new newline character to set
     
  • Argout:
    Tango::DEV_VOID     : no argout
     
  • Command allowed for:

     


16 - PurgeCommBuffer

  • Description: Purge the input and the output of the buffer of communication. This permits to have responses coherent with answers.
     
  • Argin:
    Tango::DEV_VOID     : no argin
     
  • Argout:
    Tango::DEV_BOOLEAN     : true if success
     
  • Command allowed for:
